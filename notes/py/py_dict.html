Sort the dictionary based on values

Dictionary to sort based on the values

python names = {'ZIZ': 8, 'RGY': 2, 'TYM': 1, 'SDC': 10, 'VUW': 2, 'JCQ': 2, 'PXN': 2, 'JGU': 3, 'CGB': 1, 'BHB': 10} print(names) 

{'ZIZ': 8, 'RGY': 2, 'TYM': 1, 'SDC': 10, 'VUW': 2, 'JCQ': 2, 'PXN': 2, 'JGU': 3, 'CGB': 1, 'BHB': 10}

Sort the dictionary using a lambda function

python sorted(names,key=lambda x:names[x]) 

['TYM', 'CGB', 'RGY', 'VUW', 'JCQ', 'PXN', 'JGU', 'ZIZ', 'SDC', 'BHB']

Sort the dictionary in descending order

python sorted(names,key=lambda x:names[x],reverse=True) 

['SDC', 'BHB', 'ZIZ', 'JGU', 'RGY', 'VUW', 'JCQ', 'PXN', 'TYM', 'CGB']

Sort the dictionary using a function

```python

Function to extract the value from a dictionary

def values(x): return names[x] ```

python sorted(names,key=values) 

['TYM', 'CGB', 'RGY', 'VUW', 'JCQ', 'PXN', 'JGU', 'ZIZ', 'SDC', 'BHB']

python sorted(names,key=values,reverse=True) 

['SDC', 'BHB', 'ZIZ', 'JGU', 'RGY', 'VUW', 'JCQ', 'PXN', 'TYM', 'CGB']
